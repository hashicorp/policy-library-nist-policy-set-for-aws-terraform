# EC2 - Ensure Metadata Service only allows IMDSv2

| Provider            | Category   |
|---------------------|------------|
| Amazon Web Services | Networking |

## Description

This controls ensures that Ec2 metadata service only allows IMDSv2. With IMDSv2, every request is now protected by session authentication.

This rule is covered by the [ec2-metadata-imdsv2-required](https://github.com/hashicorp/policy-library-NIST-Policy-Set-for-AWS-Terraform/blob/main/policies/ec2/ec2-metadata-imdsv2-required.sentinel) policy.
